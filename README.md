# Dedektif Sivri: Tavuk Döner Vakası

Türkiye'nin en sinir bozucu dedektifiyle 2001 yılının efsanevi Flash oyunu **Dedektif Sivri: 1001 Lezzet**'ten ilham alınan tarayıcı tabanlı point-and-click dedektif oyunu.

**[▶ Oyna](https://YOUR_USERNAME.github.io/dedektif-sivri/)**

---

## Hikaye

Kadıköy'ün arka sokaklarında faaliyet gösteren Dedektif Sivri, bir sabah beklenmedik bir vakayla karşılaşır: **Ramazan Usta'nın yirmi yıllık döneri çalınmıştır.**

Şüpheliler, ipuçları, komşu dedikodular ve sarımsaklı yoğurt lekesi sizi bekliyor.

## Oynanış

- Sahnelerdeki nesnelere ve karakterlere tıklayın
- İpuçlarını toplayın, not defterinizi doldurun
- Karakterlerle konuşun, doğru soruları sorun
- Yeterli kanıt topladıktan sonra suçluyu ilan edin

## Teknik

- Saf HTML5 + CSS + JavaScript (sıfır bağımlılık)
- Çizgi roman noir görsel stili
- 4 sahne, 5 karakter, 15+ diyalog dalı
